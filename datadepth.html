What is data depth?

Data depth can be seen as a generalization of the univariate median. Data depth measures are used in multivariate analysis to determine appropriate 'centers' in distributions of data points. The data points in a multivariate distribution can be given a depth measure which represents their similarity to the center of the distribution. Using these methods, we can generate various center outward rankings of data points.

What makes a good data depth measure?

The center given a data depth measure should have some of the key properties of the univariate median in a multivariate space. Generally, there are four main criteria that depth measure methods aim to satisfy.

Affine Invariance
The depth measures and rankings of the data points should be resistant to affine transformations. Affine transformations can be seen as both a linear transformation and a translation. If a data depth measure is affine invariant the center of a dataset will remain the center of the dataset under an affine transformation.
Maximality at the Center
The depth function should have a unique 'center' where which the depth function takes on its maximum value.
Monotonicity
The depth function should be monotone decreasing as points move away from the 'center' of the distribution.
Vanishing at Infinity
As points approach infinity, the depth function of the points should also approach zero.
